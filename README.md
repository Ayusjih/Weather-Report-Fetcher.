# Weather Report Fetcher (API-Based)

A Python command-line application that fetches real-time weather data using the OpenWeatherMap API.

## Features
- Fetches real-time temperature, humidity, and weather conditions.
- Uses OpenWeatherMap API.
- Handles errors (Invalid city, Network issues, Invalid API Key).
- Clean console-based user interface.

## Prerequisites
- Python 3.8 or higher.
- An active Internet connection.
- An OpenWeatherMap API Key.
